WP Portfolio Gallery is Awesome Filterable Portfolio Gallery type WordPress Plugin.

== Description ==
Super easy to use WP Portfolio Gallery. WP Portfolio Gallery Plugin allows to create unlimited Portfolio Items with Title, Description, Thumbnails, Portfolio URLs. With the plugin installation's a Custom Post Type named "WP Portfolio" will be created automatically. 
With the activation a Page will automatically create name "Portfolio". Assign this page to the Appearance>Menu item. You're done, all the Portfolio items will show here.

= Features Included =
* Filterable Portfolio
* WP Portfolio Gallery includes- Thumbnails, Portfolio URLs.
* No Post Description Limits.
* PrettyPhoto included
* Category based Portfolio Item
* Window resize auto height.
* Portfolio settings
* Many more are comming!!!

== Installation ==

*  From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'wp-portfolio-gallery.zip' file and upload it. Or,
*  Extract the zip file, copy and paste to the 'wp-conten/plugins/' directory of your WordPress Installation. 
*  Activate the plugin.
*  With the activation a Page will automatically create name "Portfolio". Assign this page to the Appearance>Menu item. You're done, all the Portfolio items will show here.
